# Planilhas

Para o curso da Programaria utilizamos os dados da pesquisa de **Análise de Diversidade 2022 da DataHackers** como base do projeto.

Inicialmente trabalhamos com esses dados em **Google Sheets** e com as **planilhas em Excel** disponibilizadas pela instrutora Carol (Programaria). As planilhas foram anexadas no Google Drive e, a partir delas, conduzimos as primeiras explorações e limpezas dos dados.

Em seguida, convertemos as planilhas para o formato **CSV**.

Trabalhamos no **Google Colab**, tanto lendo as planilhas quanto os arquivos CSV diretamente, e também usando esses arquivos para criar o banco de dados `status_brasil`. As planilhas (Sheets e Excel) continuam sendo a fonte principal para conferência e documentação dos dados utilizados na análise.
