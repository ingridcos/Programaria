# Planilhas

Para o curso da Programaria utilizamos os dados da pesquisa de **Análise de Diversidade 2022 da DataHackers** como base do projeto.

Inicialmente esses dados foram avaliados e mensurados no **Google Sheets** e nas **Planilhas Excel** disponibilizadas pela instrutora Carol (Programaria). As planilhas foram anexadas no Google Drive e, a partir delas, foram feitas as primeiras explorações e limpezas dos dados.

Em seguida, as planilhas foram convertidas para o formato **CSV**.

Foi utilizado como notebook o **Google Colab**, tanto para ler as planilhas quanto para gerar o arquivo CSV com a base tratada. As planilhas (Sheets e Excel) continuam sendo a fonte principal para conferência e documentação dos dados utilizados na análise.
