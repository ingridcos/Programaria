# Programaria

Este repositório foi criado para **fixar os conteúdos estudados** de todos os módulos do curso Programaria.  
Aqui vou concentrar os exercícios práticos, testes de código e anotações que me ajudam a revisar e consolidar o aprendizado.

## Organização
- **Módulo 1** – Abertura e Boas-Vindas
- **Módulo 2** – Google Sheets
- **Módulo 3** – Análise de Dados com Python
- **Módulo 4** – Análises Estatísticas com Python
- **Módulo 5** – Base de Dados (SQL)
- **Módulo 6** – Visualização de Dados (Locker Studio)
- **Módulo 7** – Aprendizado de Máquina
- **Módulo 8** – Referências e dicas para pessoas do gênero feminino (representatividade)

Cada pasta correspondente ao módulo pode conter:
- exercícios resolvidos  
- códigos de exemplo  

## Objetivo

O objetivo principal deste repositório é acompanhar minha evolução no curso, praticar versionamento com Git/GitHub e ter um lugar centralizado para consultar e revisar os exercícios sempre que necessário.
