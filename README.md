## 🛠️ Tecnologias utilizadas

### 🧠 Linguagem de Programação

* **Python** → linguagem principal para análise de dados

### 📊 Manipulação e Análise de Dados

* **Pandas** → manipulação e análise de dados estruturados
* **NumPy** → operações matemáticas e computação numérica

### 📈 Visualização de Dados

* **Matplotlib** → criação de gráficos básicos
* **Seaborn** → visualizações estatísticas mais elaboradas
* **Plotly** → gráficos interativos

### 💻 Ambiente de Desenvolvimento

* **Jupyter Notebook** → ambiente interativo para análise e testes

### 🗄️ Banco de Dados

* **SQL** → linguagem para consulta de dados
* **DBeaver** → ferramenta para gerenciamento de banco de dados

### 📊 Business Intelligence / Dashboards

* **Looker Studio** → criação de dashboards e relatórios visuais
