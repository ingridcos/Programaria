# 🌱 Programaria - Meus primeiros passos em Python para Análise de Dados

Fui contemplada com uma bolsa de estudos para o curso **Meus Primeiros Passos em Python para Análise de Dados** oferecido pela Programaria — uma organização dedicada à diversidade e inclusão no universo da tecnologia.

---

## 📚 Sobre o Curso

* **Plataforma:** Programaria
* **Foco:** Introdução ao Python aplicado à Análise de Dados
* **Projeto Desenvolvido:** Análise de Diversidade 2022
* **Base de Dados:** DataHackers - https://www.datahackers.com.br/
* **Objetivos:**

  * Aprender lógica de programação com Python
  * Manipular dados com bibliotecas como Pandas
  * Realizar análises iniciais de dados
  * Criar visualizações simples

## 🛠️ Tecnologias utilizadas

### 🧠 Linguagem de Programação

* **Python** → linguagem principal para análise de dados

### 📊 Manipulação e Análise de Dados

* **Pandas** → manipulação e análise de dados estruturados
* **NumPy** → operações matemáticas e computação numérica

### 📈 Visualização de Dados

* **Matplotlib** → criação de gráficos básicos
* **Seaborn** → visualizações estatísticas mais elaboradas
* **Plotly** → gráficos interativos

### 💻 Ambiente de Desenvolvimento

* **Jupyter Notebook** → ambiente interativo para análise e testes
* **Google Colab** → execução de notebooks na nuvem

### 🗄️ Banco de Dados

* **SQL** → linguagem para consulta de dados
* **DBeaver** → ferramenta para gerenciamento de banco de dados

### 📊 Business Intelligence / Dashboards

* **Looker Studio** → criação de dashboards e relatórios visuais
