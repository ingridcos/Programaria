# SQL

Esta pasta contém os scripts SQL utilizados no projeto **Análise de Diversidade 2022**.

- `criar_tabelas.sql`: comandos para criar as tabelas do banco.
- `consultas_exploratorias.sql`: queries usadas na análise exploratória.
- `consultas_insights.sql`: consultas finais usadas nos dashboards/relatórios.

Tabelas principais do banco: `Gerencia`, `Municipios_Brasileiros`, `Municipio_Status`  
Arquivo de banco de dados: `status_brasil`

Para executar os scripts, você pode usar o DBeaver ou outro cliente SQL conectado ao banco SQLite/PostgreSQL do projeto.
