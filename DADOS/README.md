# Pyhton (Google Collab)

Nesta pasta estão os notebooks e scripts utilizados para tratar os dados para a construção do mini-projeto **Análise de Diversidade 2022**.

O arquivo principal é o notebook `analise_dados.ipynb`, onde foi feita a **limpeza, transformação e organização completa** da base de dados utilizada no projeto. A partir desse notebook foram gerados os dados finais que alimentam o banco `status_brasil`, as análises no Colab e os Dashboards.

Os demais arquivos (`regressao_linear.ipynb`, `soma_ingredientes.ipynb`, `*.py`, etc.) correspondem a **exercícios de fixação do curso Programaria**, usados para praticar conceitos de Python, manipulação de dados e visualização, mas não fazem parte diretamente do fluxo principal da análise de diversidade.

Além da preparação dos dados, neste diretório também foi explorado os conceitos de **aprendizado de máquina** com foco em **regressão linear** avaliando o desempenho do modelo com métricas como erro quadrático médio e \(R^2\) em conjuntos de treino e teste. 

Os gráficos exploratórios e de modelo foram construídos com as bibliotecas **Matplotlib**, **Seaborn** e **Plotly**, para visualizar distribuições, relações entre variáveis e a qualidade das previsões.

O notebook principal de tratamento dos dados é o
[`analise_dados.ipynb`](https://github.com/ingridcos/Programaria/blob/main/DADOS/analise_dados.ipynb),
onde foi realizado a limpeza, transformação e análise exploratória.
